## â“ Common Questions

**Q: Do I need to refresh the page to see updates?**  
A: No! The page updates automatically in real-time.

**Q: Can I check in before the event starts?**  
A: No, the admin must start the event first.

**Q: What if I check in but leave early?**  
A: Just check out when you leave. Your hours will be calculated based on when you actually checked in and out.

**Q: Can I check out from a different device?**  
A: Yes! Check in on mobile, check out on website (or vice versa) - it all syncs.

**Q: What if I accidentally check out too early?**  
A: Contact an admin - they can manually adjust your hours if needed.

**Q: Do I see other people's check-ins?**  
A: You see the total check-in count, but only admins see individual check-in status.

### Problem: I don't see the check-in button
**Solution:** Make sure:
- Event is started (shows "Ongoing" status)
- You're registered for the event
- You're not an admin (admins use admin controls)

### Problem: Button says "Checking In..." forever
**Solution:**
- Refresh the page
- Check your internet connection
- Try again

### Problem: It says I'm already checked in but I'm not
**Solution:**
- Refresh the page
- If still showing, contact an admin

### Problem: My hours didn't record
**Solution:**
- Check the volunteer hours page
- If still missing, contact an admin
- Admin can manually add hours
